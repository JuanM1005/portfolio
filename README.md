## 🛠️ Sobre este proyecto

Este repositorio es mi portafolio personal — una SPA con scroll por anclas, diseño responsive y formulario de contacto funcional vía Formspree.

**Stack:** React 19 · TypeScript ~6 · Vite 8 · Tailwind CSS 4 · React Hook Form · Zod · Bun

**Ejecución local**

```bash
bun install
# Crea .env.local con tu endpoint de Formspree:
# VITE_FORMSPREE_ENDPOINT=https://formspree.io/f/<tu-id>
bun run dev
```

| Script | Descripción |
|--------|-------------|
| `bun run dev` | Servidor de desarrollo con HMR |
| `bun run build` | Build de producción (`tsc -b && vite build`) |
| `bun run preview` | Previsualiza el build de producción |
| `bun run lint` | Analiza el código con ESLint |
| `bun run format` | Formatea todos los archivos con Prettier |
